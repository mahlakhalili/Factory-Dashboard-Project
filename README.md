# Factory-Dashboard-Project

## معرفی پروژه

این پروژه یک داشبورد تحلیلی با استفاده از **Power BI** است که با هدف بررسی و تحلیل عملکرد یک **کارخانه کاغذ** طراحی شده است.

در این داشبورد، داده‌های مربوط به تولید، مصرف انرژی، مصرف آب، ضایعات و ماشین‌آلات کارخانه مورد بررسی قرار گرفته‌اند تا دیدی شفاف و مدیریتی از وضعیت کلی کارخانه ارائه شود.

تمرکز اصلی این پروژه بر روی موارد زیر بوده است:
- پاک‌سازی و آماده‌سازی داده‌ها
- مدل‌سازی داده‌ها در Power BI
- تعریف شاخص‌های کلیدی عملکرد (KPI)
- طراحی داشبورد مدیریتی با تمرکز بر تصمیم‌سازی

داده‌ها مربوط به **یک سال کاری کارخانه** هستند و این پروژه به‌عنوان یک نمونه‌کار عملی در حوزه **تحلیل داده و Business Intelligence** توسعه داده شده است.
